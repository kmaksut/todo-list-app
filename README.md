Electron + Svelte + Vite - Todo list application.

## Project Setup

### Development

```bash
npm run dev
```
---
> [!NOTE]
> Hot reload açık .
> ```bash
> "dev": "electron-vite dev --watch"
> ```
> Hot reload kapalı
> ```bash
> "dev": "electron-vite dev",
> ```
---
## Build

```bash
# Direct build
$ npm run build

# Windows build 
$ npm run build:win

# macOs build
$ npm run build:mac

# Linux build
$ npm run build:linux
```
